# Nicole Ortiz — Portfolio

Personal portfolio site built with Next.js, TypeScript, and Tailwind CSS.

## Stack

- [Next.js](https://nextjs.org) (App Router)
- TypeScript
- Tailwind CSS

## Running locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploying

The easiest option is [Vercel](https://vercel.com) — connect the repo and it deploys automatically on every push to main.
